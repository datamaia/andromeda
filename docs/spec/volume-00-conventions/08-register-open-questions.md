# 08 — Register: Open Questions

Open questions and PENDING VALIDATION items, consolidated from the `99-volume-register.md`
files of Volumes 1–14. Global `OQ-NNN` identifiers are minted here; the "Raised by" column
records every raising volume with its local register identifier, so each local entry resolves
to exactly one global row. Every `PENDING VALIDATION` occurrence in the corpus corresponds to
a row in this register (occurrences in ADR bodies resolve through the row tracking that ADR's
review condition). An open question never blocks authoring: the affected design defines a
stable abstraction and continues.

Where several volumes raised the same underlying question, consolidation merged them into one
row listing all raisers; no local identifier was renamed. Status values: `Open` | `Resolved`
(with the resolving decision named).
